## Quick Reference

**File Locations:**
- JS: `blocks/weekly-engagement/weekly-engagement.js`
- CSS: `blocks/weekly-engagement/weekly-engagement.css`
- Example: `engagement-weekly.html`

**Key CSS Classes:**
- `.weekly-engagement-container` - Main wrapper
- `.engagement-table` - Table element
- `.status-badge` - Status pill
- `.health-score` - Health display
- `.engagement-row.expanded` - Highlighted row

**Key Functions:**
- `getWeekIdentifier()` - Determine current week
- `parseTableData(rows)` - Convert table to objects
- `applyFilters()` - Filter table rows
- `createTableRow(item)` - Build table row
