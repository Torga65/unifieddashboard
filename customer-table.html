About This Table

This table displays all customer data with interactive features including search, filtering, and sorting.

Features

  • Search - Type to search across all fields
  • Filter by Status - Production, Pre-Production, Planning, etc.
  • Filter by Engagement - Active, At Risk, Critical
  • Filter by Health - Healthy, Needs Attention, Critical
  • Sort - By name, health score, or engagement level
  • Click Rows - Click any row to highlight

Columns

  • Company Name - Customer organization
  • Status - Production stage
  • Engagement - Engagement level
  • Health Score - Numeric score with visual bar (0-100)
  • Summary of Engagement - Detailed notes and activities
  • Blockers - Current obstacles or issues
  • Feedback - Customer sentiment
  • Last Updated - Date of last update

Responsive Design

Desktop: All columns visible

Tablet: Most columns visible (Feedback and Last Updated hidden)

Mobile: Essential columns only (Company Name, Status, Engagement, Health Score)
